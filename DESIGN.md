# Design System & UI/UX Guidelines - Treaty

## 1. Visual Aesthetics & Direction
The UI/UX completely avoids corporate banking or standard university portal layouts. It utilizes a high-contrast, gamified look and feel optimized for high student engagement.

## 2. Color Palette Matrix (Default Light Theme)
- Primary Background: `#F8FAFC` (Slate 50)
- Surface Cards: `#FFFFFF` (Pure White)
- Treaty Green (Primary Activation): `#22C55E` / `#00C853`
- Deep Purple (Secondary Structure): `#6D28D9` / `#7C4DFF`
- Accent Orange (Urgency Signals): `#F97316` / `#FDC400`

## 3. Typography
- Font Family: Poppins (Bold, Semibold, Medium, Regular)
- Elements use strict Tailwind class mappings for clean responsiveness.

## 4. Layout Boundaries
- Mobile-First Shell: Strict max-width constraint wrapper (`max-w-md mx-auto min-h-screen bg-slate-50 shadow-xl`).
